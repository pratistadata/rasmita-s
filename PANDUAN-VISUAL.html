Cara Admin Membuka Game

1

Buka Admin Panel

Dari landing page, klik link kecil "Admin" di bagian bawah. Kode akses: FAM2026

2

Pilih Game yang Ingin Dibuka

Di Admin Panel, terlihat daftar 6 game. Masing-masing ada tombol "Buka Game" / "Tutup Game"

3

Klik "Buka Game"

Di lobby semua pemain, game card berubah dari ⌛ MENUNGGU menjadi ✓ DIBUKA

4

Pemain Tap Card = Masuk Game

Semua pemain di lobby tinggal tap card game yang sudah terbuka untuk langsung mulai bermain

⚠️ Catatan Penting
  • Buka game satu per satu sesuai urutan acara
  • Tutup game sebelumnya sebelum membuka yang baru (agar fokus)
  • Admin Panel berjalan di device admin, pemain di device masing-masing
  • Tidak ada sinkronisasi real-time — setiap pemain punya state sendiri

🚀 Deploy & Publish Aplikasi

Pilih platform hosting gratis — Netlify paling mudah untuk pemula

🌐
DIREKOMENDASIKAN
Netlify
Drag & drop, gratis, URL instan dalam 30 detik
  1. Buka netlify.com → Sign up
  2. Di dashboard, scroll ke bawah
  3. Drag & drop file index.html
  4. Tunggu 30 detik → URL siap! ✅
  5. Opsional: ganti nama jadi rasmita-2026
URL contoh: rasmita-2026.netlify.app
🐙
UNTUK DEVELOPER
GitHub Pages
Perlu akun GitHub, gratis, URL permanen
  1. Buat repo baru di GitHub
  2. Upload index.html
  3. Settings → Pages → Deploy from main
  4. Tunggu 1-2 menit ✅
URL: username.github.io/nama-repo
▲
TERCEPAT
Vercel
Deploy dalam hitungan detik, performa terbaik
  1. Buka vercel.com → Sign up
  2. Add New Project → Upload
  3. Upload folder berisi index.html
  4. Klik Deploy ✅
URL: rasmita-2026.vercel.app
