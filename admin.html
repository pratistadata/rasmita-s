📽️ Panduan Setup Proyektor & TV
✅ Konsep utama: Game berjalan di browser. Admin buka di laptop → proyektor/TV tampilkan ke layar besar. Peserta main dari HP masing-masing.
Opsi Koneksi
🔌

Opsi A — Kabel HDMI (Paling Stabil)

1. Colok HDMI dari laptop ke proyektor/TV
2. Laptop: Display Setting → Extend/Duplicate
3. Pindah browser ke layar proyektor
4. Tekan F11 untuk fullscreen
5. Peserta scan QR atau buka URL di HP

📡

Opsi B — Screen Mirroring Wireless

Windows: Win+K → pilih TV (Miracast)
Mac: Menu bar → AirPlay → pilih Apple TV
Android: Smart View/Screen Cast
iPhone: AirPlay ke Apple TV
⚠️ Pastikan HP & TV di WiFi yang sama

🌐

Opsi C — Deploy Online (Terbaik)

1. Upload ke Netlify/Vercel (gratis)
2. Dapatkan URL publik
3. Admin buka URL di laptop → proyektor
4. Peserta buka URL yang sama di HP
✅ Tidak perlu satu jaringan WiFi

Alur Acara dengan Proyektor
1

Sebelum Acara Dimulai (H-30 menit)

Nyalakan proyektor/TV. Hubungkan laptop. Buka rasmita-family-gathering-2026.html di browser Chrome/Edge. Fullscreen dengan F11. Pastikan semua foto keluarga sudah diupload di Admin → Foto Keluarga.

2

Peserta Masuk (Registrasi)

Tampilkan halaman landing di proyektor. MC umumkan: "Buka browser di HP kalian, ketik URL ini, lalu masukkan nama dan kode room FAM2026". Jika deploy online, tampilkan QR code URL-nya.

3

Buka Game Satu Per Satu

Di Admin Panel ini, buka game satu per satu via tab Kontrol Game. Setelah klik "Buka Game", peserta langsung bisa klik game tersebut dari lobby HP masing-masing.

4

Saat Game Berlangsung

Layar proyektor tetap di halaman lobby atau game yang sedang aktif. MC membaca aturan dari ATURAN-GAME.md. Untuk Game 3 (Voting), tunjukkan hasil di proyektor setelah semua vote masuk.

5

Umumkan Pemenang

Setiap game selesai, tampilkan leaderboard di proyektor. Untuk Game 6 (Tebak Lagu), layar sing-along sangat bagus ditampilkan di layar besar agar semua bisa ikut bernyanyi bersama!

Tips Teknis
Browser terbaik: Google Chrome atau Microsoft Edge (bukan Safari untuk audio Game 6)
Resolusi proyektor: Set ke 1920×1080 untuk tampilan terbaik
Audio Game 6: Sambungkan speaker eksternal ke laptop untuk suara lebih keras
Jika layar hitam: Tekan F11 lagi, atau cek kabel HDMI
Backup plan: Screenshot semua game sebagai cadangan jika internet mati

🎭 Game 5 — Database Komedian
Database komedian sudah tersedia di dalam kode game (200 komedian). Di sini kamu bisa lihat dan upload foto untuk komedian pilihan. Foto bersifat opsional — jika tidak diupload, muncul emoji 🎭.

🎵 Game 6 — Setting Lagu
Game 6 berjalan dengan melodi sintetis otomatis (tidak perlu file apapun). Untuk kualitas audio lebih baik, upload file MP3 intro lagu. Jika file tidak ada, otomatis fallback ke sintetis.
# Judul Artis Potong di Mode Audio Upload MP3
✂️ Panduan Potong Audio
Tool gratis online: online-audio-cutter.com atau mp3cut.net

Langkah:
1. Download lagu dari sumber legal (Spotify, Apple Music, etc.)
2. Buka online-audio-cutter.com → Upload file
3. Potong sesuai timestamp di tabel atas
4. Export sebagai MP3
5. Rename file sesuai kolom "Nama File"
6. Upload di tombol "Upload MP3" di tabel atas
⚠️ Setelah upload MP3, kamu perlu edit 2 baris kode di rasmita-family-gathering-2026.html untuk mengaktifkan mode MP3. Lihat panduan di README.md.
🎮 Buka / Tutup Game
Peserta hanya bisa masuk game yang sudah kamu buka. Buka game satu per satu sesuai urutan acara untuk mengatur flow gathering.
🃏 Game 1 — Memory Match Foto
Max Pemain
2–4
Durasi
15 mnt
Min Foto
8
Game ini membutuhkan foto anggota keluarga. Pastikan sudah upload minimal 8 foto di Foto Keluarga. Game membuat 8–15 pasang kartu dari foto yang ada.
1

Mulai Giliran

Satu pemain memegang HP. Tap kartu pertama → foto terbalik.

2

Tap Kartu Kedua

Jika cocok → +10 poin, giliran berlanjut. Jika tidak cocok → kedua kartu tertutup, giliran berpindah.

3

Selesai

Semua pasangan ditemukan. Pemain poin tertinggi = pemenang.

❌ Tidak boleh menunjuk/memberi tahu posisi kartu ke pemain lain. ✅ Boleh berteriak kegirangan!
Script MC:
"Ingat-ingat posisi kartu yang sudah dibuka sebelumnya! Itu kunci menang game ini. Yang ingatannya bagus, waktunya unjuk gigi! 4 orang bergantian ya — 1 HP dibagi bersama."
Setup: Kumpulkan 2–4 orang per meja/grup. Satu orang pegang HP, yang lain nonton dan ingat posisi kartu.

Foto anggota keluarga ditampilkan blur/terpotong, perlahan terlihat. Tebak lebih cepat = poin lebih besar. Semakin banyak foto yang diupload, semakin seru gamenya.

Foto Muncul Buram

Foto anggota keluarga tampil tersembunyi. Tiap pemain lihat di HP masing-masing.

2

Poin Menurun

400–500 poin jika tebak saat masih blur. Makin lama menunggu, makin kecil poinnya (min 100).

3

Ketik Nama

Tulis nama anggota keluarga di kolom jawaban. Ejaan tidak harus tepat — nama panggilan diterima.

❌ Dilarang tanya ke orang lain. ❌ Dilarang lihat HP orang lain. ✅ Nama panggilan boleh.
Script MC:
"Foto pertama selalu yang paling susah karena masih blur total. Tapi kalau kamu kenal betul anggota keluarga ini, pasti bisa tebak dari rambut atau bentuk muka aja! Jangan tunggu foto jelas — rugi poin!"
