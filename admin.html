📽️ Panduan Setup Proyektor & TV
✅ Konsep utama: Game berjalan di browser. Admin buka di laptop → proyektor/TV tampilkan ke layar besar. Peserta main dari HP masing-masing.
Opsi Koneksi
🔌

Opsi A — Kabel HDMI (Paling Stabil)

1. Colok HDMI dari laptop ke proyektor/TV
2. Laptop: Display Setting → Extend/Duplicate
3. Pindah browser ke layar proyektor
4. Tekan F11 untuk fullscreen
5. Peserta scan QR atau buka URL di HP

📡

Opsi B — Screen Mirroring Wireless

Windows: Win+K → pilih TV (Miracast)
Mac: Menu bar → AirPlay → pilih Apple TV
Android: Smart View/Screen Cast
iPhone: AirPlay ke Apple TV
⚠️ Pastikan HP & TV di WiFi yang sama

🌐

Opsi C — Deploy Online (Terbaik)

1. Upload ke Netlify/Vercel (gratis)
2. Dapatkan URL publik
3. Admin buka URL di laptop → proyektor
4. Peserta buka URL yang sama di HP
✅ Tidak perlu satu jaringan WiFi

Alur Acara dengan Proyektor
1

Sebelum Acara Dimulai (H-30 menit)

Nyalakan proyektor/TV. Hubungkan laptop. Buka rasmita-family-gathering-2026.html di browser Chrome/Edge. Fullscreen dengan F11. Pastikan semua foto keluarga sudah diupload di Admin → Foto Keluarga.

2

Peserta Masuk (Registrasi)

Tampilkan halaman landing di proyektor. MC umumkan: "Buka browser di HP kalian, ketik URL ini, lalu masukkan nama dan kode room FAM2026". Jika deploy online, tampilkan QR code URL-nya.

3

Buka Game Satu Per Satu

Di Admin Panel ini, buka game satu per satu via tab Kontrol Game. Setelah klik "Buka Game", peserta langsung bisa klik game tersebut dari lobby HP masing-masing.

4

Saat Game Berlangsung

Layar proyektor tetap di halaman lobby atau game yang sedang aktif. MC membaca aturan dari ATURAN-GAME.md. Untuk Game 3 (Voting), tunjukkan hasil di proyektor setelah semua vote masuk.

5

Umumkan Pemenang

Setiap game selesai, tampilkan leaderboard di proyektor. Untuk Game 6 (Tebak Lagu), layar sing-along sangat bagus ditampilkan di layar besar agar semua bisa ikut bernyanyi bersama!

Tips Teknis
Browser terbaik: Google Chrome atau Microsoft Edge (bukan Safari untuk audio Game 6)
Resolusi proyektor: Set ke 1920×1080 untuk tampilan terbaik
Audio Game 6: Sambungkan speaker eksternal ke laptop untuk suara lebih keras
Jika layar hitam: Tekan F11 lagi, atau cek kabel HDMI
Backup plan: Screenshot semua game sebagai cadangan jika internet mati

Selesai

Setup: Kumpulkan 2–4 orang per meja/grup. Satu orang pegang HP, yang lain nonton dan ingat posisi kartu.
